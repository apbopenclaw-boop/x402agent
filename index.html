How It Works

Three steps. No signup, no API keys, no vendor lock-in.

1

Request

Your AI agent sends a standard HTTP request to any paid endpoint.

2

Pay

The server returns HTTP 402. Your agent signs a USDC micropayment on Base.

3

Receive

Re-send with the payment header — structured data returned instantly.

Frequently Asked Questions

What is x402agent?▼

x402agent.no is Norway's AI agent API gateway — a collection of Norwegian data APIs that any x402-compatible AI agent can pay for and use instantly. No API keys, no accounts, no subscriptions. Just HTTP requests and USDC micropayments.

What is the x402 payment protocol?▼

x402 is an open protocol that lets AI agents pay for API access using cryptocurrency micropayments. When an agent hits a paid endpoint, it receives an HTTP 402 Payment Required response with pricing details. The agent signs a USDC payment on the Base network and resends the request — all programmatically, in a single request cycle.

How much do the APIs cost?▼

Pay-per-query pricing starts at $0.005 per request. Each endpoint has its own price clearly shown in the 402 response. Payments are in USDC on the Base network with sub-cent gas fees. No minimums, no commitments.

Do I need an API key or account?▼

No. All x402agent APIs are completely keyless. Any AI agent with a USDC wallet on Base can start making queries immediately. Free endpoints (like /health) require nothing at all.

Where does the data come from?▼

Each service uses official Norwegian data sources: Brønnøysundregistrene (company data), Statistics Norway/SSB (economic statistics), Kartverket (mapping and property), Patentstyret (intellectual property), MET Norway (weather), and Common Crawl (SEO/backlinks).

Which AI agents support x402?▼

Any agent that can sign EIP-712 typed data and send HTTP requests can use x402. Coinbase's AgentKit, Langchain with x402 middleware, and custom agents using the x402 Python or TypeScript SDKs all work out of the box.
