How It Works

Three steps. No signup, no API keys, no vendor lock-in.

1

Request

Your AI agent sends a standard HTTP request to any paid endpoint.

2

Pay

The server returns HTTP 402. Your agent signs a USDC micropayment on Base.

3

Receive

Re-send with the payment header — structured data returned instantly.

# 1. Agent requests data $ curl https://norwegian-companies-house.fly.dev/company/923609016 # → HTTP 402 Payment Required # → x-payment-requirements: {"price":"0.01","currency":"USDC","network":"base"} # 2. Agent signs payment, retries with header # → HTTP 200 OK {"name":"EQUINOR ASA","type":"ASA","employees":21376,...}
